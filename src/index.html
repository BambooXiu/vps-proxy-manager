使用教程

快速开始

1

配置 VPS 连接

进入设置页面，填入你的 VPS 信息：

  • 主机地址（IP 或域名）
  • SSH 端口（默认 22）
  • 用户名（通常为 root）
  • 认证方式：密码或 SSH 密钥

点击"测试连接"确认能正常连接到 VPS

2

配置 IPRoyal 代理

在设置页面下方填入 IPRoyal 代理信息：

  • 代理地址（如 proxy.iproyal.com）
  • 端口号
  • 用户名和密码

IPRoyal 账号可在 iproyal.com 注册获取

3

一键部署

进入部署页面，点击"开始部署"按钮。

系统将自动完成以下操作：

  • 安装 Xray-core
  • 生成 VLESS + Reality 配置
  • 配置 IPRoyal 和直连两种模式
  • 启用 BBR 拥塞控制优化
  • 配置防火墙规则

部署过程约需 2-5 分钟，请耐心等待

4

获取客户端配置

部署完成后，进入客户端页面：

  • 扫描二维码导入配置（推荐）
  • 复制 VLESS 链接导入
  • 复制完整 JSON 配置导入 v2rayN
5

模式切换

在仪表盘页面可以随时切换代理模式：

  • IPRoyal 模式：流量通过 IPRoyal 代理出口，IP 显示为 194.50.146.79
  • 直连模式：流量直接从 VPS 出口，IP 显示为 VPS 地址

常见问题

部署失败怎么办？

常见原因及解决方法：

  • VPS 系统不是 Debian/Ubuntu - 建议使用 Ubuntu 20.04+
  • SSH 连接不稳定 - 检查网络，尝试使用密钥认证
  • 端口被占用 - 确保 443 端口未被其他服务占用
  • 内存不足 - 建议至少 512MB 内存
如何更新 Xray 版本？

连接 VPS 后，SSH 执行以下命令：

bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install

更新后需重启：systemctl restart xray

连接速度慢怎么优化？

优化建议：

  • 确保 BBR 已启用（部署时自动配置）
  • 选择距离较近的 VPS 节点
  • VLESS + Reality + Vision 节点请关闭 MUX
  • 避免高峰时段使用
IPRoyal 和直连模式有什么区别？
  • IPRoyal 模式：出口 IP 为 194.50.146.79（美国 Delaware），适合需要特定地区 IP 的场景
  • 直连模式：出口 IP 为 VPS 本身的 IP，速度更快，适合普通访问
