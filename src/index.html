使用教程

快速开始

1

配置 VPS 连接

进入设置页面，填入你的 VPS 信息：

  • 主机地址（IP 或域名）
  • SSH 端口（默认 22）
  • 用户名（通常为 root）
  • 认证方式：密码或 SSH 密钥

点击"测试连接"确认能正常连接到 VPS

2

配置 ISP 代理

在设置页面下方填入你购买的 ISP 代理信息：

  • 代理地址（如 198.51.100.10）
  • 端口号
  • 用户名和密码

请以代理服务商后台提供的地址、端口、用户名和密码为准

3

一键部署

进入部署页面，点击"开始部署"按钮。

系统将自动完成以下操作：

  • 安装 Xray-core
  • 生成 VLESS + Reality 配置
  • 配置 ISP 代理和直连两种模式
  • 启用 BBR 拥塞控制优化
  • 配置防火墙规则

部署过程约需 2-5 分钟，请耐心等待

4

获取客户端配置

部署完成后，进入客户端页面：

  • 扫描二维码或复制 VLESS 链接只会导入节点参数，适合普通连接测试
  • 重新生成配置：部署信息变更、重新部署或需要刷新微信直连和 DNS 优化时使用，会根据当前 VPS 参数重新生成 VLESS 链接和完整 JSON
  • 复制完整配置：复制包含微信直连、国内直连、DNS 分流和 MUX 关闭设置的完整 JSON，推荐用于 v2rayN/v2rayNG 系列客户端

完整 JSON 导入步骤：

  1. 在客户端页面点击重新生成配置，确认没有报错。
  2. 点击复制完整配置（含微信直连优化）。
  3. 新建一个文本文件，粘贴刚复制的内容，保存为 novabit-proxy.json；如果系统先保存成 .txt，请把后缀改为 .json。
  4. 电脑使用：在 v2rayN 桌面端选择本地导入/从文件导入配置，选择刚保存的 novabit-proxy.json。
  5. 手机使用：把 novabit-proxy.json 通过 AirDrop、数据线、网盘或聊天工具发送到手机，在 v2rayNG 中选择本地导入/从文件导入并选中该 JSON。
  6. 导入后启用新配置；如果列表里有旧配置，优先选择名称最新或重新导入的完整配置。

微信直连和 DNS 优化只存在于完整 JSON 中，单独扫描二维码或导入 VLESS 链接不会带上这些规则。

5

模式切换

在仪表盘页面可以随时切换代理模式：

  • ISP 代理模式：流量通过已配置的 ISP 代理出口，IP 显示为代理服务商出口
  • 直连模式：流量直接从 VPS 出口，IP 显示为 VPS 地址

常见问题

部署失败怎么办？

常见原因及解决方法：

  • VPS 系统不是 Debian/Ubuntu - 建议使用 Ubuntu 20.04+
  • SSH 连接不稳定 - 检查网络，尝试使用密钥认证
  • 端口被占用 - 确保 443 端口未被其他服务占用
  • 内存不足 - 建议至少 512MB 内存
如何更新 Xray 版本？

连接 VPS 后，SSH 执行以下命令：

bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install

更新后需重启：systemctl restart xray

连接速度慢怎么优化？

优化建议：

  • 确保 BBR 已启用（部署时自动配置）
  • 选择距离较近的 VPS 节点
  • VLESS + Reality + Vision 节点请关闭 MUX
  • 避免高峰时段使用
微信或国内访问慢怎么办？

通常是客户端只导入了节点参数，微信直连和 DNS 优化没有生效。解决步骤：

  • 不要只扫描二维码或导入 VLESS 链接；这两种方式只包含节点参数，不包含国内直连和 DNS 分流规则。
  • 进入客户端页面，先点击重新生成配置，再点击复制完整配置。
  • 把复制的内容保存为 novabit-proxy.json，在 v2rayN/v2rayNG 中选择本地导入/从文件导入该完整 JSON。
  • 导入后启用新配置，确认正在使用完整 JSON 配置，而不是旧的二维码或 VLESS 链接配置。
  • 完整 JSON 内置微信直连和 DNS 优化，微信、腾讯和国内访问会优先直连，国外及 OpenAI 相关流量仍按代理规则处理。
ISP 代理和直连模式有什么区别？
  • ISP 代理模式：出口 IP 为当前配置的代理服务商出口，适合需要特定地区 IP 的场景
  • 直连模式：出口 IP 为 VPS 本身的 IP，速度更快，适合普通访问
